TUDO PRONTO! Apenas 4 comandos para colocar seu projeto no GitHub!

═══════════════════════════════════════════════════════════════════════════════

PASSO 1: FAZER COMMIT (Terminal)
────────────────────────────────

cd /home/anderson.nasario/Documentos/Nasario/jira-dash
git add .
git commit -m "feat: Jira Dashboard v1.0 com modais e filtros de projetos"

═══════════════════════════════════════════════════════════════════════════════

PASSO 2: CRIAR REPOSITÓRIO (GitHub Web)
────────────────────────────────────────

1. Abra: https://github.com/new
2. Name: jira-dash
3. Description: Jira Dashboard com filtros de projetos, análise de dados e modais
4. Visibility: Public (ou Private)
5. Clique: Create repository

COPIE A URL que GitHub vai mostrar (algo como):
https://github.com/SEU_USUARIO/jira-dash.git

═══════════════════════════════════════════════════════════════════════════════

PASSO 3: CONFIGURAR REMOTE (Terminal)
──────────────────────────────────────

git remote add origin https://github.com/SEU_USUARIO/jira-dash.git
git branch -M main
git push -u origin main

⚠️ TROCAR "SEU_USUARIO" POR SEU USUÁRIO DO GITHUB!

═══════════════════════════════════════════════════════════════════════════════

PASSO 4: VERIFICAR (Browser)
─────────────────────────────

Abra: https://github.com/SEU_USUARIO/jira-dash

Deve mostrar seus arquivos e a mensagem do commit.

✅ PRONTO! Seu projeto está no GitHub! 🎉
