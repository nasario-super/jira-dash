───────────────────────────────

cd /home/ssm-user/projetos

───────────────────────────

cd jira-dash


PASSO 3: Executar script de deploy
──────────────────────────────────

                            ❓ TROUBLESHOOTING
================================================================================

Se der erro "Cannot find package":
> npm install

Se build falhar:
> rm -rf node_modules package-lock.json
> npm install
> npm run build

Se PM2 não funcionar:
> npm install -g pm2
> pm2 startup
> pm2 save

Para ver detalhes do erro:
> pm2 logs jira-dashboard
