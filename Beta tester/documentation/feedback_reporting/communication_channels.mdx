# Communication Channels for Beta Testers

This document provides an overview of the available communication channels for beta testers, focusing on how to provide feedback and report issues effectively. This guide is part of the beta testing documentation and aims to facilitate smooth communication between testers and developers.

## How to Report Issues

When encountering an error during testing, follow these steps to report it effectively:

1. **Capture a Screenshot**: Take a screenshot of the issue.
2. **Gather Logs**: Open DevTools (F12) and navigate to the Console tab to gather logs.
3. **Collect Information**:
   - Browser and Version
   - Selected Projects
   - Active Tab (e.g., Overview, Daily, Velocity, Alerts)
   - Description of what you were doing when the issue occurred

4. **Submit the Report**:
   - Include the screenshot, logs, and collected information.
   - Use the following format for the report:
     ```
Title: [BUG] Brief Description
Description: Steps to reproduce + Expected vs Actual Outcome
     ```
